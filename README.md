
# CineFind

CineFind is a personal project to showcase my React skills. The app lets users search for movies and TV shows, view details, and explore trending content.


## Features

- **Keyword Search**: Search for movies or TV Shows by typing a keyword in the search bar.  
- **Media Details**: View detailed information about movies or series.
- **Language Switch**: Easily switch between English and French.
- **Light/Dark Mode**: Choose between a light or dark theme for e better viewing experience.
- **Trending Content**: Display trending movies and shows directly on the home page.


## Tech Stack

**React:** Main framework used to build the application

**TypeScript:** For strong typing and more robust code

**Tailwind CSS and DaisyUI:** For design and UI components

**React Router:** Handling navigation between pages

**Axios**: fetching data from an external API

**TMDB API**: Movie and TV Shows data is provided by [The Movie Database (TMDB)](https://www.themoviedb.org/)


## Setup

Clone the repository

```bash
git clone https://github.com/Fr4nckie/cine-find.git
cd cine-find
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Other useful script

- `npm run build` : Build the project for production.

- `npm run preview` : Preview the built version locally.

- `npm run lint` : Check the code using ESLint.
